# Курсовая работа: "Todo & Notes"

## Описание проекта

Веб-приложение для управления задачами и заметками. Проект реализован на Python с использованием Flask и SQLite. Приложение позволяет создавать, просматривать, редактировать и удалять задачи и заметки, а также отмечать задачи как выполненные.

## Ссылка на гит

https://github.com/OneDeadL1ne/TodoAndNotes

## Основные возможности

- Страница `Todo` для создания и управления списком задач.
- Страница `Notes` для добавления и хранения заметок.
- Удаление задач и заметок.
- Отметка задачи как выполненной.
- Статистика по задачам и заметкам.

## Технологии

- Python 3.10+
- Flask
- SQLite
- HTML/CSS

## Запуск проекта

### 1. Установка зависимостей

Откройте терминал в папке `app` и выполните:

```powershell
py -3 -m pip install -r requirements.txt
```

### 2. Запуск приложения

```powershell
py -3 app.py
```

### 3. Открытие в браузере

Перейдите по адресу:

```text
http://127.0.0.1:5000
```

## Структура проекта

- `app/app.py` — основной файл Flask-приложения.
- `app/requirements.txt` — зависимости проекта.
- `app/templates/` — HTML-шаблоны.
- `app/static/style.css` — стили приложения.
- `app/db/database.sql` — файл со схемой базы данных.
