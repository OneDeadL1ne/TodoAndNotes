# ER-диаграмма

## Описание

ER-диаграмма описывает основные сущности и связи предметной области проекта "Todo & Notes".

```mermaid
erDiagram
    USERS {
        INTEGER id PK "Первичный ключ"
        TEXT username "Имя пользователя"
    }
    NOTES {
        INTEGER id PK "Первичный ключ"
        INTEGER user_id FK "Внешний ключ пользователя"
        TEXT title "Заголовок заметки"
        TEXT body "Текст заметки"
        TEXT created_at "Дата создания"
    }
    TODOS {
        INTEGER id PK "Первичный ключ"
        INTEGER user_id FK "Внешний ключ пользователя"
        TEXT title "Текст задачи"
        TEXT status "Статус задачи (active/done)"
        TEXT created_at "Дата создания"
    }

    USERS ||--o{ NOTES : "создает"
    USERS ||--o{ TODOS : "владеет"
```

## Сущности

- `USERS` — пользователи системы.
- `NOTES` — заметки, связанные с пользователем.
- `TODOS` — задачи, связанные с пользователем.

## Связи

- Один пользователь может создавать несколько заметок.
- Один пользователь может иметь несколько задач.
